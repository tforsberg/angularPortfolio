This site was built from scratch using PHP. The database that powers the site is on a MySQL server hosted by AWS.

Code design

As much as possible the site was built with the DRY principle in mind. In order to reduce code repetition, the pages were built on a template that imports repeated elements such as the header and footer. The only part of any given page that changes is the main content. Everything else is found in a central location.

UI Design

Much like the portfolio you're on right now, the UI was inspired by Material Design by Google. Everything from colours to layout to typography was lifted and transferred over to both my portfolio and this e-commerce site. Material Design is brilliant and easy to understand. It was a no brainer to use it.

Security:

Some of the security considerations in building this site were password hashing, SQL injection protection, and end-to-end encryption.

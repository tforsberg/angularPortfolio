Java Object Oriented Crazy Eights

Java

Status: Complete

Link to Github

This application was an assignment at KPU INFO 2313 wherein all object oriented principles learned were to be demonstrated.

It is a game of crazy eights played through the command line. A user can play against a computer, or two computers can be pitted against eachother

In order to play the game, the player is shown the contents of their hand and asked to play a card by entering a number or to draw a new card.

The first player to have an empty hand wins!

Code design

This project features core object oriented principles:

  • Inheritance
  • Polymorphism
  • Abstraction
  • Encapsulation
